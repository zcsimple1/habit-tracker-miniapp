# 习惯打卡系统 - 设计预览

这是习惯打卡系统的静态页面设计预览，用于在开发前验证 UI/UX 设计效果。

## 📱 页面列表

### 1. 首页 - 分类模式
**文件**: `index-category.html`

- 按分类分组显示打卡项目
- 每个分类下按时间排序
- 有固定时间的项目排前面，无时间的排后面
- 点击"打卡"按钮可以切换状态

**预览**:
```
┌─────────────────────┐
│  今日打卡            │
│  📋 分类模式 | 时间模式│
├─────────────────────┤
│  🏢 工作             │
│    ☐ 08:00 早会     │
│    ☐ 09:30 提交周报  │
│    ☐ 写日报         │
├─────────────────────┤
│  ❤️ 健康             │
│    ☐ 07:00 晨跑     │
│    ☐ 健身           │
└─────────────────────┘
```

---

### 2. 首页 - 时间模式
**文件**: `index-time.html`

- 所有项目直接按时间排序
- 每项前面显示分类图标
- 适合按时间流程逐项打卡

**预览**:
```
┌─────────────────────┐
│  今日打卡            │
│  📅 时间模式 | 分类模式│
├─────────────────────┤
│  [❤️] 07:00 晨跑    │
│  [👶] 07:30 送孩子  │
│  [🏢] 08:00 早会    │
│  [🏢] 09:30 提交周报│
│  [🏢] 写日报        │
│  [❤️] 健身          │
└─────────────────────┘
```

---

### 3. 已完成的打卡
**文件**: `hidden.html`

- 显示今日已打卡的项目
- 按分类分组
- 可以取消打卡记录（带确认）
- 显示打卡时间

**预览**:
```
┌─────────────────────┐
│  已完成的打卡   [返回]│
├─────────────────────┤
│  ❤️ 健康             │
│    ✓ 晨跑 (07:05)   │
│    ✓ 喝水 (15:30)   │
├─────────────────────┤
│  🏢 工作             │
│    ✓ 早会 (08:02)   │
└─────────────────────┘
```

---

### 4. 编辑打卡项目
**文件**: `habit-edit.html`

- 项目名称编辑
- 所属分类选择
- 循环规则设置（每日/工作日/周末/每周指定）
- 固定打卡时间设置
- 有效期设置

---

### 5. 统计中心
**文件**: `stats.html`

- 总体概况（完成率、打卡次数、活跃天数）
- 完成率排行榜
- 完成趋势图表
- 按分类统计
- 分享报告按钮

---

### 6. 分享卡片
**文件**: `share-card.html`

包含三种分享卡片模板：
1. **连续打卡成就** - 展示坚持 N 天的成就
2. **月度总结** - 本月完成率、最佳项目、最长连续天数
3. **分类成就** - 某分类下的所有项目完成情况

---

## 🎨 设计特点

### 1. 视觉设计
- **渐变背景**: 使用紫色渐变，现代感强
- **圆角卡片**: 柔和的视觉效果
- **阴影效果**: 层次分明
- **图标分类**: 使用 Emoji 图标，直观易懂

### 2. 交互设计
- **状态切换**: 点击打卡按钮切换状态
- **模式切换**: 分类模式 ⇄ 时间模式
- **取消确认**: 取消打卡时弹出确认框
- **悬停效果**: 鼠标悬停时的视觉反馈

### 3. 响应式设计
- 固定宽度 375px（iPhone 尺寸）
- 适合在浏览器中预览
- 可直接移植到小程序

## 📝 设计说明

### 预设分类（9个）
- 🏢 工作 (蓝色)
- ❤️ 健康 (红色)
- 📚 学习 (紫色)
- 🏠 家庭 (橙色)
- 👶 孩子 (绿色)
- 💰 财务 (青色)
- ⭐ 个人成长 (粉色)
- 💬 社交 (橙色)
- 📌 其它 (灰色)

### 排序规则
1. **分类模式**: 先按分类分组，分类内按时间排序
2. **时间模式**: 所有项目直接按时间排序，显示分类图标

### 时间显示
- 有固定时间: 显示具体时间（如 08:00）
- 无固定时间: 显示 "--:--" 或 "无固定时间"
