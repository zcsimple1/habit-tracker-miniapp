# PRD - 习惯打卡 + 轻量统计（MVP）

## 目标
做一个可以快速上线、可持续迭代的小程序：用户创建习惯 → 每天打卡 → 看到连续天数与完成率。

## 用户故事
- 作为用户，我想创建一个习惯（例如：跑步），并每天打卡。
- 作为用户，我想看到我坚持了多少天（连续 streak）。
- 作为用户，我想看到近 7 天、本周、本月的完成情况。

## MVP 功能
### 1. 习惯管理
- 创建习惯：名称
- 编辑习惯：名称
- 删除习惯：同时删除其日志

### 2. 打卡
- 每个习惯每天最多一次打卡（布尔）
- 支持取消打卡

### 3. 轻量统计
- streak：从今天往前连续完成天数
- 近 7 天完成率：done/7
- 本周完成率：从周一至今日 done/total
- 本月完成天数：从月初到今日完成天数

## 数据结构（本地存储）
```js
state = {
  habits: [{ id, name, createdAt, updatedAt }],
  logs: {
    [habitId]: {
      [ymd]: true
    }
  }
}
```

## 页面
- Tab1: 打卡页（习惯列表 + 今日打卡按钮）
- 习惯表单页（新建/编辑/删除）
- Tab2: 统计页（每个习惯一张卡片显示统计）

## 迭代方向（V2+）
- 云开发：登录、多端同步
- 提醒：订阅消息 + 本地提醒
- 自定义频率：每周 x 次、每周几
- 图表：周/月趋势折线
- 导出：CSV / 图片海报
