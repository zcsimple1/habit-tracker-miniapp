# Habit Tracker Mini Program (习惯打卡 + 轻量统计)

一个微信小程序练手项目：习惯管理、每日打卡、连续天数、周/月统计。

## 功能（MVP）
- 习惯列表：创建/编辑/删除
- 每日打卡：按天记录是否完成
- 统计：
  - 当前连续天数（streak）
  - 近 7 天完成率
  - 本月完成天数

## 技术方案
- 前端：微信小程序原生（miniprogram）
- 后端：可选（本项目先做本地存储版）
- 存储：`wx.setStorageSync` 本地持久化

## 目录结构
- `miniprogram/` 小程序源码
- `docs/` 设计与 PRD

## 快速开始
1. 用微信开发者工具导入项目目录（选择 `miniprogram/` 作为小程序目录，或直接导入根目录并配置）。
2. 运行并体验。
